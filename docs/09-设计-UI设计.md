# UI设计：页面与交互规范

## 设计目标

UI 设计服务于 PDF 要求的三条主线：创作者一站式创作、AI 审核与质量反馈、内容榜单分发。产品需要清晰区分 C 端内容前台和 B 端创作者后台：C 端强调内容消费与榜单浏览，B 端强调高效创作、审核反馈和分发管理。

## 页面信息架构

C 端内容前台：

模仿小红书网站设计

- 登录注册:为弹窗形式，样式效果以及要求可以借鉴 B端项目的登录注册页面设计，保持视觉和交互一致性。
- 首页信息:头部有搜索框,下部有 tap 选择类型,默认展示全部内容,以瀑布流的形式展现。
- 内容详情页：展示发布者、发布时间、正文、配图。
- 榜单:(实现滚动加载)
  - 热点榜：支持依据内容质量分、阅读热度、发布时间等多维度进行综合排序。
  - 爆文榜：支持依据内容质量分、阅读热度、发布时间等多维度进行综合排序。
- 用户中心：个人信息,我的作品。

B 端创作者后台：

- 登录注册：邮箱或手机号登录注册，支持退出登录。
- 创作工作台：草稿、已发布内容、审核状态、质量分概览。
- AI 创作编辑器：主题输入、Prompt 选择、素材选择、AI 生成候选、图文编辑器、保存状态、审核入口。
- Prompt 管理：系统 Prompt 与个人 Prompt 的创建、编辑、使用次数。
- 素材管理：素材上传、列表、标签、基础合规状态。
- 内容管理：草稿、审核中、需改写、已发布、已更新内容。
- 审核结果：风险等级、风险片段、审核理由、质量评分、一键合规改写。
- 分发中心：查看内容在热点榜、爆文榜、推荐榜中的表现和排序解释。

## 核心页面原型重点

首页信息流：

- 首屏直接展示内容列表，不做营销型大 Hero。
- 顶部提供热点榜、爆文榜、推荐榜入口。
- 内容卡片展示标题、摘要、封面、作者、发布时间和阅读热度。

榜单页：

- 使用标签页切换热点榜、爆文榜、推荐榜。
- 内容卡片展示标题、摘要、封面、作者、发布时间、质量分、阅读热度。
- 卡片尺寸稳定，图片固定比例，避免滚动时布局抖动。
- 无限滚动需要展示加载中、无更多内容和缓存更新时间。

内容详情页：

- 正文阅读优先，标题、作者、发布时间和封面层级清晰。
- 展示阅读、点赞、分享等基础互动。
- 推荐相关内容或返回榜单入口，但不干扰正文阅读。

创作工作台：

- 顶部展示创作入口和关键指标。
- 中部展示最近草稿、审核中内容、已发布内容。
- 侧边或顶部导航保持简洁，突出“创作、Prompt、素材、内容、审核、分发”。

AI 创作编辑器：

- 左侧或顶部为创作参数：主题、受众、风格、关键词、Prompt、素材。
- 中间为图文编辑器。
- 右侧为 AI 候选区、审核结果和质量建议。
- 保存状态必须可见：本地已保存、云端同步中、离线编辑中、存在冲突。

审核结果页：

- 用明确状态区分通过、需改写、驳回。
- 风险片段高亮，展示原因和建议。
- 质量评分以总分 + 维度条展示。
- 一键合规改写按钮只在需要改写时突出。

分发中心：

- 展示单篇内容的阅读量、点赞量、分享量、质量分和榜单排名。
- 展示排序解释，说明质量分、阅读热度、发布时间对排名的贡献。
- 支持从内容管理跳转查看发布后表现。

## 视觉风格

- C 端风格：清爽、内容优先、阅读友好，突出信息流和榜单效率。
- B 端风格：克制、信息密度更高、偏创作者生产工具。
- 色彩：以中性背景和高对比文本为主，AI 操作可使用品牌强调色，风险状态使用语义色。
- 语义色：通过为绿色，需改写为橙色，驳回为红色，离线/同步为蓝色或灰色。
- 字体：中文内容阅读优先，标题和正文层级清晰，避免过大的装饰标题。
- 卡片：只用于内容项、结果面板和弹窗，不把整页套进大卡片。

## 交互状态

- AI 生成：loading、成功、结构校验失败、服务超时、重试。
- 草稿保存：本地已保存、云端同步中、云端已同步、离线编辑、同步失败、冲突。
- 审核：审核中、通过、需改写、驳回。
- 发布：未审核不可发布、审核通过可发布、发布成功、发布失败。
- 榜单：首屏加载、加载更多、无更多内容、缓存更新时间。

## 响应式规则

- C 端移动端优先单列信息流，桌面端可使用双列或三列内容网格。
- B 端桌面端优先三栏编辑器、表格 + 筛选 + 详情抽屉布局。
- 平板端将 AI 面板折叠为右侧抽屉。
- 移动端优先单列：参数、编辑器、AI 结果按流程上下排列。
- 榜单页移动端使用单列卡片，桌面端可双列或三列。
- 所有按钮文字不能挤压溢出，关键操作保留图标和明确文案。

## 原型工具建议

- 快速生成：Motiff AI、Pixso AI、Figma AI。
- 二次加工：Figma 或 Pixso。
- 代码原型：v0 生成 React + Tailwind + shadcn/ui 页面，再迁移到项目。
- 设计参考：Semi Design、Ant Design、头条号/抖音创作者中心、小红书创作者服务平台。
